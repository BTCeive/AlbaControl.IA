Estructura creada

- `src/pages/` : Carpetas para cada página de la app (HOME, Nuevo albarán, Borradores, Historial, Proveedores/Productos, Opciones).
- `config/` : Documentación y placeholders de build/manifest.
- `storage/` : Carpetas para almacenar datos del usuario (drafts, completed, providers, products, config, backups).
- `app/` : Esqueleto mínimo Android con `MainActivity.kt` y layout con botones de navegación (placeholders).
- `.gitignore` y `README.md` en la raíz.

GitHub y Play Store:
- El repositorio está pensado para versionarse en GitHub. Para publicar en Play Store será necesario el scaffold completo, un keystore y configuración del certificado.
